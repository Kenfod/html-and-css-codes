Flexbox

        Flexbox is short for the Flexible Box Layout module.

        It is a layout method for arranging items in rows or columns.

        It makes it easier to design a flexible responsive layout structure, without using float or positioning.

        A flexbox always consists of:

            #1. A Flex Container - the parent (div container)  element.
            #2. Flex Items - the items inside the div container.

        Flexbox vs. Grid
        The CSS Flexbox Layout should be used for one-dimensional layout, with rows OR columns.

        The CSS Grid Layout should be used for two-dimensional layout, with rows AND columns.
