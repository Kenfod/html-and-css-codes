Transformations

        The transform property in CSS is used to change the coordinate space of the visual formatting model.

        This property lets you rotate, scale, skew, or translate an element.
        
        With the CSS transform property you can use the following 2D transformation functions:

            -translate()
            -rotate()
            -scaleX()
            -scaleY()
            -scale()
            -skewX()
            -skewY()
            -skew()
            -matrix()
