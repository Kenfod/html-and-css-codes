Display

The display property specifies if and how an element should be displayed.

    The display property can be:

    block=level - which starts on a new line and takes up the full width available. 
    (Examples include and not limited to h1, div, p, form, header, footer)

    inline - Do not start on a new line and it's width is limited to what is need.
    (Examples include and not limited to span, a, img)
